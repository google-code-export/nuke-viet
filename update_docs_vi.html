- Sau khi nâng cấp xong vào quản lý giao diện => thiết lập layout nhấp Lưu lại để hệ thống cập nhật giao diện mới cho chức năng xem danh sách thành viên.
- Từ r1841 hệ thống tạo thêm chức năng xem danh sách thành viên và đặt phân quyền ở mức quản trị được xem. Để thay đổi quyền xem, vào cấu hình module của module users và thay đổi giá trị đó thành Tất cả hoặc Thành viên.
